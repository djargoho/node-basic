노드에서는 블로킹, 논블로킹, 동기와 비동기라는 네 용어가 혼용된다. 중요한 개념. 

동기와 비동기: 함수가 바로 return 되는지 여부. 
블로킹과 논블로킹 : 백그라운드 작업 완료 여부. 

노드의 방식 
1) 동기-블로킹 방식 : 
    백그라운드 작업 완료 여부를 확인, 호출한 함수가 바로 return 되지 않고 백그라운드 작업이 끝나야 return 된다. 
2) 비동기-논블로킹 방식 :
    호출한 함수가 바로 return, 다음 작업으로 넘어가며,
    백그라운드 작업 완료 여부는 신경쓰지 않고 나중에 백그라운드에서 알림을 줄 때 처리한다. 
그 외에 방식은 없다고 보면 된다.
